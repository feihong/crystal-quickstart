# Feihong's Crystal quickstart

## Installation

    sudo apt-get install gcc pkg-config git tzdata \
                           libpcre3-dev libevent-dev libyaml-dev \
                           libgmp-dev libssl-dev libxml2-dev
    sudo snap install crystal --classic

## Commands

Build and run program:

    crystal hello.cr
